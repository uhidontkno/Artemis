Artemis

Discord's Best FOSS
Server Protection Bot.


Artemis is Discord's best FOSS server protection bot for blocking VPNs, Tor, malicious IPs and alt accounts. With multiple client-side and server-side verification layers, you can keep your ban evaders at bay.


Invite GitHub



Finally a verification bot without bulls**t.

Artemis is designed to respect everyone's privacy. That means only storing data that is absolutely necessary to the use of Artemis and hashing & encrypting your data to be inaccessible from outsiders. Artemis is exactly what it says on the tin, no extra features you don't need, no privacy invading features, and complete control over your data. Our privacy policy is easily accessible when you run /privacy and it is written in simple language for complete transparency.





No clunky web dashboard required

Setup can be done in less than 10 seconds and if you want to put the easiest challenge on yourself, you can unplug your mouse and set up Artemis using only your keyboard because setup is a single slash command away.
